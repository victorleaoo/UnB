Relatório para o Laboratório WEB

Passos para execução

  1. Instalação do servidor Apache: $ sudo apt install apache2
  2. Verificar se o apache está ligado: $ ps -axu | grep apache
  3. Caso não, iniciar: $ /etc/init.d/apache2 start
  4. Com o serviço ligado, testar na web: http://localhost
  5. Dessa forma, é possível criar novos arquivos html no diretório: /var/www/html
  6. As novas páginas html podem ser acessadas na web, então.
