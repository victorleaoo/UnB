Relatório para o Laboratório SMTP

Passos para execução

  1. Inicialmente, é necessário instalar o pacote postfix: $ sudo apt-get install postfix
  2. Assim, com ele instalado, pode-se começar a configuração com as seguintes entradas para as etapas:
    3.  Ok
     Internet Site
     NONE
     mail.jvgv.com
     mail.jvgv.com, localhost.jvgv, localhost
     No
     127.0.0.0/28, 192.168.10.0/24
     Yes
     0
     +
     all
  3. É interessante dar um reload no serviço: $ systemctl reload postfix
  4. Com isso, é possível iniciar o serviço postfix: $ /etc/init.d/postfix start
  5. Adiciona-se os usuários da rede: $ adduser 192.168.10.50
  6. Então, é possível enviar um e-mail para usuários cadastrados: $ mail -v 192.168.10.50@jvgv.com
  7. Por fim, para verificar se o e-mail chegou, o comando a seguir mostra o arquivo criado quando um e-mail é chegado com sucesso: $ cat /var/mail/192.168.10.50
