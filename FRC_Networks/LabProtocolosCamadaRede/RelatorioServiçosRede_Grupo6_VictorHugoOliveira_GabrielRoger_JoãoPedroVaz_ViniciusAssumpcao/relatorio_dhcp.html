Relatório para o Laboratório DHCP

Passos para execução

  1. Primeiramente, é necessário instalar o servidor DHCP: sudo apt-get install isc-dhcp-server -y
  2. Em seguida, deve-se configurar o DHCP server no arquivo presente em: /etc/dhcp3/dhcpd.conf
  3. As mudanças importantes a serem feitas são:
    4.  option domain-name "www.jvgv.com";
     option domain-name-serves 192.168.10.2;
     subnet 192.168.10.0 netmask 255.255.255.0{
      range 192.168.10.50 192.168.10.100;
      option routers 192.168.10.2;
     }
  4. Com isso, é possível inicializar o DHCP server: /etc/init.d/isc-dhcp-server start
  5. É possível verificar os logs do DHCP com o seguinte comando: tail -f /var/log/syslog
